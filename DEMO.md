# 🎬 CopyFlow Demo - 完整演示

> 从输入主题到生成可发布素材的完整流程演示

---

## 📝 演示场景

**主题**：AI 如何改变内容创作

**目标**：为公众号/小红书/知乎生成一套完整的内容素材

---

## 🚀 Step 1: 运行命令

```bash
python copyflow.py run --topic "AI 如何改变内容创作"
```

**执行过程**：
```
[1/3] 生成多风格文案...
  ✓ 数据驱动型
  ✓ 故事叙述型
  ✓ 观点输出型
  ✓ 热点追踪型
  ✓ 知识科普型
  ✓ 创意脑洞型
  ✓ 商务专业型
  ✓ 励志鸡汤型
  ✓ 反思质疑型
  ✓ 轻松幽默型

[2/3] 生成配图...
  ✓ 为 10 套文案生成匹配配图

[3/3] 整理素材文档...
  ✓ 生成 content-review.md

✅ 完成！输出目录：./output/
```

**耗时**：约 3 分钟

## ✅ Step 3: 审核与发布

### 审核流程

1. **打开 content-review.md**
   - 查看 10 套文案
   - 选择最适合的 2-3 套

2. **修改润色**
   - 根据品牌调性调整
   - 补充具体案例
   - 优化标题和标签

3. **选择配图**
   - 查看 images/ 目录
   - 如果不满意，可以重新生成

### 发布到平台

**公众号**：
- 复制文案到公众号编辑器
- 上传配图
- 调整排版

**小红书**：
- 使用故事型/轻松幽默型文案
- 配图 + 标签
- 发布

**知乎**：
- 使用数据驱动型/知识科普型文案
- 补充更多细节
- 发布

## 💡 使用技巧

### 1. 主题选择

**好的主题**：
- ✅ "AI 如何改变内容创作"（具体场景）
- ✅ "远程办公的 5 个效率工具"（明确数量）
- ✅ "从 0 到 1 做个人品牌"（清晰路径）

**不好的主题**：
- ❌ "AI"（太宽泛）
- ❌ "一些想法"（不明确）
- ❌ "今天的感悟"（太主观）

### 2. 风格选择

根据平台特性选择：

| 平台 | 推荐风格 |
|------|---------|
| 公众号 | 数据驱动型、知识科普型、商务专业型 |
| 小红书 | 故事叙述型、轻松幽默型、励志鸡汤型 |
| 知乎 | 数据驱动型、反思质疑型、知识科普型 |
| 抖音 | 热点追踪型、创意脑洞型、轻松幽默型 |

### 3. 配图优化

如果生成的配图不满意：

```bash
# 重新生成特定风格的配图
python copyflow.py generate-images \
  --config ./output/copies.json \
  --output ./output/images \
  --styles "数据驱动型,故事叙述型"
```

### 4. 批量生产

```bash
# 为多个主题批量生成
for topic in "主题1" "主题2" "主题3"; do
  python copyflow.py run --topic "$topic" --output "./output/$topic"
done
```
